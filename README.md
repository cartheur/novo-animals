# novo-animals

A new generation design of animals in the format of a retro-computer consisting of 26 TTL chips in the following map:
```
                                                   ...
          |--------------->S->|------->F=}->video   ^
          |  H->|  +-+        |  +-+     ^         GPU
          |->V->|->|R|        |  |R|  |->G->audio   v
          |        |A|<>|->B->|->|O|->|            ...
          |->X->|->|M|  |     |  |M|  |             ^
          |->Y->|  +-+  | fn->|  +-+  |->C->[ECU]   :
          |--------->A->|     |{=HL<--|            CPU
          |     Rx->Ei->|     |<-PC<--|->Eo->Tx     :
          |          I->|     |<-Pg<--|->O          v
          |___________________________|            ...
```
